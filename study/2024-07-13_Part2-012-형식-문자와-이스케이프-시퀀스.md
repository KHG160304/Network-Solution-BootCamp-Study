# printf 함수

- 형식을 가진 문자를 출력할때 사용할 수 있는 함수이다.



| 형식문자 | 자료형                     | 형식문자 | 자료형   |
| -------- | -------------------------- | -------- | -------- |
| `%c`     | int, char                  | %d       | int      |
| %o       | int                        | %u       | unsigned |
| %x, %X   |                            | %e, %E   |          |
| %f       | double                     | %g       | double   |
| %s       | String, char 배열          | %zd      | size_t   |
| %lld     | 64비트 정수, long long int | %l64u    |          |



# 이스케이프 시퀀스 종류

| 문자 | 의미            | 문자 | 의미                |
| ---- | --------------- | ---- | ------------------- |
| \a   | 경고음          | \\\  | Back-Slash 문자자체 |
| \b   | Backspace       | \?   |                     |
| \f   | Form feed       | \\'  |                     |
| \n   | New line        | \\"  |                     |
| \r   | Carriage return | \ooo | 8진수               |
| \t   | Tab             | \xhh | 16진수              |
| \v   | Vertical Tab    |      |                     |
