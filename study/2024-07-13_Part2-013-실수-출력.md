# 실수를 출력하는 방법

```cpp
int main(void)
{
    // 실수 자료형이 차지하는 메모리의 크기를 출력
    printf("%d\n", sizeof(123.456F)); // float 형식 4byte
    printf("%d\n", sizeof(123.456));  // double 형식 8byte
    
    //float 형식의 실수를 출력
    printf("%f\n", 123.456F);         
    
    //double 형식의 실수를 출력
    // 소수점 아래 자리는 기본 6자리까지 표시한다. 
    // 소수점 아래 자리 숫자가 6자리 이하이면, 남는 자리는 0으로 채운다.
    // %g 형식문자는 0으로 채우는 행위를 하지 않는다.
    // 윈도우 환경에서는 long double을 지원하지 않는다.
    printf("%f\n", 123.456);  // double     , 출력 결과 123.456000
    printf("%lf\n", 123.456); // long double, 출력 결과 123.456000
    printf("%g\n", 123.456);  // long double, 출력 결과 123.456
}
```



```cpp
double dData = 123.456;

// 정수값 123을 출력한다. 전체 자리수는 4자리 이고 남는 자리는 0으로 채워라
printf("%04d\n", 123);

// 소수점 아랫자리는 1자리까지 출력해라. (2째 자리에서 반올림 시켜라.)
printf("%.1f\n");
// 소수점 아랫자리는 2자리까지 출력해라. (3째 자리에서 반올림 시켜라.)
printf("%.2f\n");
// 소수점 아랫자리는 3자리까지 출력해라. (4째 자리에서 반올림 시켜라.)
printf("%.3f\n");

// 총 소수점 포함해서 12자리로 출력해라, 단 소수점은 3자리까지 출력해라(4째 자리에서 반올림 시켜라.)
printf("%12.3f\n");
// 위와 같은데, 12 자리중 남는 자리는 0으로 채워라
printf("%012.3f\n");

```
